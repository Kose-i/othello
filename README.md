----|----

----|----

## How to Use

`config`内の`detect_hand.txt`ファイルを編集して,`player` `computer`の色を決めます.

`mkdir build`

`cd $_`

`cmake ..`

`make`

実行ファイルができたことを確認して実行します.

`./main`

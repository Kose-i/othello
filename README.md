| Hyoukakansuu | for computer class |
| Player | player class |
| Point | position class |

## How to Use

`config`内の`detect_hand.txt`ファイルを編集して,`Black` `White`のプレーヤーの名前を決めます.

`mkdir build`

`cd $_`

`cmake ..`

`make`

実行ファイルができたことを確認して実行します.

`./main`
